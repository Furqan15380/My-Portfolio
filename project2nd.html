Digital Marketing Course

Role: Frontend Developer

Overview

A high-converting, professionally designed landing page for a Digital Marketing Course. This project focuses on a modern, user-centric interface designed to drive student engagement and registrations. It features a fully responsive layout, ensuring a seamless experience across mobile, tablet, and desktop devices.

Key Features

  • Dynamic Hero Section: A compelling landing area with a modern registration form and key course highlights to capture leads effectively.
  • Responsive Layout: Developed with a mobile-first approach using Bootstrap 5, ensuring optimal performance on all screen sizes.
  • Key Highlights Section: Showcases course value propositions such as industry certifications, live doubt-clearing sessions, and downloadable content using custom SVG icons.
  • Career Pathways: A dedicated section illustrating various career opportunities in the digital marketing field (SEO, Content Marketing, etc.).
  • Trust Building: Includes a "Get Certified" section with professional certification previews to build authority and trust with potential students.
  • Interactive UI Elements: Implementation of custom CSS animations, hover effects, and Bootstrap modals for a premium look and feel.

Technologies Used

Core: HTML5, CSS3 (Custom Styling)
Framework: Bootstrap 5
Icons: Font Awesome & Custom SVGs
Typography: Google Fonts (Oswald, Noto Sans)
